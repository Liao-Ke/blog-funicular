# 一级标题

段落之间用空行隔开。

第二段。*斜体*、**粗体**、和 `等宽字体`。无序列表像这样：

- 这个
- 那个
- 另一个

注意 —— 也别管星号 —— 实际文字从第四列开始。

> 引用块这样写。
>
> 如果愿意，可以跨越多个段落。

使用三个短横线表示破折号。使用两个短横线表示范围（比如 "第12--14章"）。三个点... 会变成省略号。支持 Unicode。☺

## 二级标题

有序列表：

1. 第一项
2. 第二项
3. 第三项

注意again实际文字从左起第四列开始。代码示例：

    # 让我重复一下 ...
    for i in 1 .. 10 { do-something(i) }

正如你猜的，缩进四个空格。顺带说，不用缩进块，也可以用定界块：

```
define foobar() {
    print "Welcome to flavor country!";
}
```

（这样复制粘贴更方便）。可以给定界块加语言标记让语法高亮：

```python
import time
# 快速数到十！
for i in range(10):
    #（但别太快）
    time.sleep(0.5)
    print i
```

### 三级标题

嵌套列表：

1. 首先，准备这些材料：

    - 胡萝卜
    - 芹菜
    - 扁豆

2. 烧水。

3. 把所有东西倒进锅里，按这个步骤：

    找到木勺
    打开锅盖
    搅拌
    盖上锅盖
    把木勺危险地架在锅把上
    等十分钟
    回到第一步（或者关火）

    别撞到木勺，不然会掉。

注意again文字总是对齐在四个空格的缩进（包括上面第三项的延续行）。

这里是 [一个网站](http://foo.bar) 的链接，一个 [本地文档](local-doc.html)，以及 [本文中的一个章节链接](#二级标题)。脚注这样写 [^1]。

[^1]: 脚注内容在这里。
