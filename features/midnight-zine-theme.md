# 午夜手札主题

## 目标

为博客增加一套更有辨识度的新视觉主题，让首页、文章卡片、侧栏和文章页从默认 Material 风格转向“编辑型个人博客”的纸张、油墨与品红强调色体验。

## 修改范围

- `src/config.ts`：调整默认主题色相。
- `tailwind.config.cjs`：扩展正文与标题字体栈。
- `src/layouts/Layout.astro`：启用全局字体族。
- `src/styles/variables.styl`：重设主题色、背景色、卡片色、边框、阴影、按钮和文本 token。
- `src/styles/main.css`：增加全局背景纹理、焦点状态、减少动态效果适配，以及卡片和按钮质感。

## 核心实现

- 使用品红主色、暖橙与青色辅助色，避免全站单一色调。
- 用细线边框、轻阴影和纸张网格纹理替代原本更重的 MD3 卡片观感。
- 标题优先使用中文衬线/宋体字体栈，正文保留 Roboto 与常见中文无衬线字体回退。
- 保留现有色相滑杆，用户仍可调整主色 hue。
- 增加 `prefers-reduced-motion` 处理，降低动效敏感用户的负担。

## 影响范围

主题 token 会影响全站共用组件，包括导航栏、文章卡片、侧栏、搜索面板、分页、文章正文容器、上一篇/下一篇按钮和浮层面板。文章内容、路由、数据结构和构建流程未改变。

## 验证方式

- 运行 `pnpm check` 验证 Astro 类型与模板。
- 运行 `pnpm format` 与 `pnpm lint` 处理格式和代码检查。
- 运行 `pnpm build` 验证静态构建和 Pagefind 索引生成。
- 启动本地开发服务器后检查桌面与移动端页面显示。

## 已知限制

主题使用系统中文字体栈，不新增字体依赖；不同操作系统上的字体细节会略有差异。背景中的 `color-mix()` 依赖现代浏览器支持，旧浏览器会退化为基础背景色。
